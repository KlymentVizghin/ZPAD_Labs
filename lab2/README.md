# Лабораторна робота №2: Наука про дані (підготовчий етап) 

## Опис завдання
Лабораторна робота присвячена основам збору, очищення та базової обробки даних за допомогою Python та бібліотеки Pandas. Робота складається з двох частин:
1. **Частина 1:** Автоматизоване завантаження даних VHI-індексу для областей України за допомогою модуля `urllib`. Очищення даних, заміна індексів областей та створення функцій для формування вибірок.
2. **Частина 2:** Аналіз даних електроспоживання домогосподарств. Профілювання часу виконання скриптів за допомогою `timeit`, складна фільтрація даних, нормалізація, стандартизація, обчислення коефіцієнтів кореляції Пірсона і Спірмена, а також застосування One-Hot Encoding.

## Вимоги до системи
- **ОС:** Ubuntu / Linux (або інша сумісна ОС)
- **Інтерпретатор:** Python 3.8+
- **Пакетний менеджер:** pip

## Інструкція з налаштування

1. **Створення віртуального середовища:**
   Відкрийте термінал у папці з лабораторною роботою (`lab2`) і виконайте:
   python3 -m venv venv
   source venv/bin/activate
   pip install --no-cache-dir -r requirements.txt
## Отримання датасетів

    Частина 1: Файли NOAA VHI завантажуються автоматично під час запуску коду в part1.ipynb та зберігаються у папку data/. Реалізовано механізм запобігання повторному завантаженню.

    Частина 2: Датасет необхідно завантажити вручну з UCI Machine Learning Repository. Архів треба розпакувати, а файл household_power_consumption.txt покласти у папку data/ всередині директорії лабораторної.
    
## Інструкція для запуску

    Активуйте віртуальне середовище:

    source venv/bin/activate

    Запустіть сервер Jupyter Notebook:

    jupyter notebook

    У браузері, що відкриється, запустіть файл part1.ipynb або part2.ipynb.

    Виконуйте скрипти послідовно
