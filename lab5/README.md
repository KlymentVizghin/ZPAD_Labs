# Лабораторна робота №5: Наука про дані: обмін результатами та початковий аналіз

**Опис завдання:** Створення веб-додатку за допомогою Streamlit для візуалізації та аналізу даних вегетаційних індексів (VCI, TCI, VHI) з Лабораторної роботи №2.

**Вимоги до середовища:**
- ОС: Linux (Kali)
- Python 3.9+

**Встановлення залежностей:**
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
**Запуск додатку:**
streamlit run app.py
