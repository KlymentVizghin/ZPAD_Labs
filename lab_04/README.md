# Лабораторна робота №4: Візуалізація даних 2

## Опис завдання
Програма дозволяє візуалізувати графік гармоніки з накладеним шумом. Користувач має можливість через інтерактивний інтерфейс (слайдери, чекбокси) змінювати амплітуду, частоту, фазу гармоніки, а також середнє значення та дисперсію шуму. 
Додатково реалізовано фільтрацію зашумленого сигналу з можливістю регулювання частоти зрізу фільтра для наближення сигналу до чистої гармоніки.

## Вимоги до системи
* **OS:** Linux (Kali / Ubuntu).
* **Interpreter:** Python 3.9+
* **Пакети:** Вказані у `requirements.txt` (numpy, scipy, matplotlib).

## Інструкція зі встановлення та запуску

1. Переконайтеся, що ви знаходитесь у директорії `lab_04`.
2. Створіть та активуйте віртуальне середовище:
   python3 -m venv venv
   source venv/bin/activate
3. Встановіть необхідні залежності:
   pip install -r requirements.txt
4. Запустіть скрипт:
   python main.py
## Інструкція користувача
* **Amplitude, Frequency, Phase:** Змінюють параметри базової гармоніки. При їх зміні конфігурація згенерованого шуму не змінюється (шум залишається "прикріпленим" до сигналу).
* **Noise Mean, Noise Cov:** Змінюють математичне сподівання та дисперсію шуму. При цьому генерується новий масив шуму.
* **Filter Cutoff:** Змінює частоту зрізу ФНЧ. Чим менше значення, тим сильніше згладжується графік (допомагає знайти ідеальне значення для відновлення чистого сигналу).
* **Чекбокс "Show Noise":** Вмикає/вимикає відображення шуму (дозволяє подивитись на "чисту" гармоніку).
* **Чекбокс "Show Filtered":** Вмикає/вимикає відображення відфільтрованої кривої.
* **Кнопка "Reset":** Повертає всі повзунки до початкових значень.
