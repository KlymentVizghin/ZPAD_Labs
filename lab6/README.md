# Лабораторна робота №6: C++ openCV

## Вимоги до системи
- **ОС:** Kali Linux
- **Компілятор:** GCC
- **Білд система:** CMake 3.10+
- **Залізо:** Наявність веб-камери для захоплення відео.

## Встановлення та запуск
1. Встановлення залежностей (OpenCV, CMake, компілятор):
   ./preinstall.sh
2. Збірка проєкту:
   ./build.sh
3. Запуск програми:
   ./run.sh
## Управління
1 - Нормальний режим
2 - Інверсія кольорів
3 - Фільтр Canny
q або ESC - Вихід
