# Лабораторна робота №3: Візуалізація даних

## Опис завдання
Обрано датасет Automobile з UCI. Проведено очищення даних від пропущених значень. Побудовано 6 графіків для аналізу залежностей (scatter plots, гістограма з кастомними діапазонами, багатовимірні візуалізації).

## Вимоги до системи
- ОС: Kali Linux
- Python: 3.9+

## Інструкція зі встановлення та запуску
1. Клонувати репозиторій та перейти в гілку (або завантажити `lab_03`).
2. Створити віртуальне середовище:
   `python3 -m venv venv`
3. Активувати віртуальне середовище:
   `source venv/bin/activate`
4. Встановити залежності:
   `pip install -r requirements.txt`
5. Для отримання датасету завантажте файл imports-85.data з сайту https://archive.ics.uci.edu/dataset/10/automobile і покладіть у корінь папки `lab_03`.
6. Запустити Jupyter Notebook:
   `jupyter notebook`
7. Відкрити файл `Lab3.ipynb` та виконати комірки.
